# Clothing Store Frontend

Frontend React/Vite cho dự án Clothing Store.

Xem hướng dẫn cài đặt và chạy toàn bộ hệ thống tại [README chính](../README.md).

## Chạy riêng frontend

```bash
npm install
npm run dev
```

Frontend mặc định chạy tại `http://localhost:5173` và proxy `/api` đến backend `http://localhost:8080`.
